# Quantum-error-recovery-
Quantum error recovery implementation in MATLAB and Python. 


## MATLAB Dependencies

- MATLAB R2023a+
- CVX 2.2 (Build 9, a0a77590)
- Install CVX from [cvxr.com](https://cvxr.com/cvx/doc/install.html)

## Python Installation

cd qer
pyenv install 3.12.4
pyenv virtualenv 3.12.4 qer_env
pyenv local qer_env
pip install -e ./python
